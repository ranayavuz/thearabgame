Game Manual

Welcome to the manual for running game.py. Follow these steps to get started:

Installation

  1. Install Python (version 3.10 or higher) using Self Service or from the official website.
  2. Open your terminal and run the following command to install the required dependency: 
    pip3 install colorama

Running the Game

  1. Keep the terminal open and type the following command, leaving a space at the end: 
    python3
  2. Drag your downloaded file into the terminal and press Enter to run it.

Note: Ignore the __pycache__ folder. It contains Python's binary files used for importing configurations.

Configuration

Before running the game, make sure to configure the config.py file. Do not change the variable names!

  • black_terminal: Set this to True if your terminal's background is black, otherwise set it to False.
  • mode: Set the game's difficulty. Available options are:
    • "baby"
    • "easy"
    • "normal"
    • "hard"
    • "dragon"

More configuration options will be added in future updates!
